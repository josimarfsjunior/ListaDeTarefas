# ListaDeTarefas

Projeto criado com objetivo de construir uma lista de tarefas diárias.

## Regras Criadas:

### Campos Obrigatórios

1 - Tarefa a ser realizada: (Exemplo: Estudar Javascript hoje);

2 - Botão Registrar

### Função principais

1 - Adicionar Tarefa (Lista);

2 - Renderizar Tarefa;

3 - Excluir Tarefa;

4 - Salvar Localmente as tarefas (Local Storage);

## Experiência alcançada

Desenvolvi uma lista de tarefas diárias utilizando HTML, CSS e JavaScript, permitindo ao usuário adicionar e excluir tarefas de forma dinâmica. Durante o desenvolvimento, aprendi a manipular o DOM, capturar eventos, atualizar a interface em tempo real por meio de uma função de renderização que exibe a lista sempre que uma tarefa é adicionada ou removida e organizar melhor a lógica da aplicação.

Também adquiri conhecimento sobre o uso do **Local Storage**, possibilitando armazenar as tarefas no navegador e mantê-las disponíveis mesmo após fechar ou atualizar a página. Além disso, aprimorei minhas habilidades em estilização com CSS, estruturação com HTML e versionamento de código utilizando Git e GitHub para publicação do projeto.
